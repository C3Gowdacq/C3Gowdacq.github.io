Dalal Street AI — NSE Stock Market AnalyzerFeatured
AI-powered stock analysis platform using LangGraph to orchestrate multiple agents (fundamental, sentiment, momentum, risk) analyzing NSE NIFTY 500 stocks and generating BUY/SELL/HOLD signals.
PythonFastAPILangGraphLangChainGemini 2.5FinBERTReactTypeScriptRedisChromaDB
 Agentic AI ↗
What it does
An AI-powered stock analysis platform for NSE NIFTY 500 stocks. Multiple specialized LangGraph agents run in parallel — fundamental analysis, FinBERT-based news sentiment from Indian publishers, momentum/OHLCV signals, and risk scoring — all producing a unified BUY/SELL/HOLD recommendation.
Architecture
LangGraph orchestrates 4 parallel agents → Google Gemini 2.5 for reasoning → async httpx pipeline for live OHLCV & financial metrics → FinBERT for news sentiment → Redis caching on FastAPI → React + TypeScript dashboard with candlestick charts and indicators. ChromaDB stores vectorized financial context.

 Deep Learning ↗

Parkinson's Disease Detection Using SVM
ML pipeline using Support Vector Machines to classify Parkinson's disease from patient biomedical vocal features. Includes preprocessing, feature selection, and visualisation.
PythonScikit-learnSVMMatplotlib
 Machine Learning ↗
What it does
Binary classification of Parkinson's disease vs healthy patients using vocal frequency biomarkers (MDVP features). Achieves high accuracy on the UCI Parkinson's dataset with proper train/test splitting and cross-validation.
Approach
Data cleaning → feature scaling (StandardScaler) → SVM with RBF kernel → hyperparameter tuning via GridSearchCV → feature importance via permutation analysis → confusion matrix and ROC curve visualisation.
